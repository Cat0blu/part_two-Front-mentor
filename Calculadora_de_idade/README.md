# PROJECT: Aplicativo de calculadora de idade#

## Sobre: 

- calculadora de idade, que verifica datas validas usando o Date(). 

## Ferramentas utilizada

    - Html(semantica)
    - Css e sass(estilos)
    - Node.js(instalação de ferramentas)
    - js(interatividade)
